# Editing Passes

## Developmental Edit

Check promise, structure, audience, scope, argument/story, missing chapters, weak causality, unsupported claims, and continuity.

## Line Edit

Check paragraph flow, rhythm, emphasis, scene clarity, transition, dialogue, and sentence-level force.

## Copy Edit

Check consistency, grammar, spelling, terms, citations, style guide, figure/table references, and permissions markers.

## Proofread

Check final typos, formatting, page references, captions, headings, and repeated errors.
