# Visual Hybrid Book Writing

## Workflow

Design the page or spread as the core unit. Decide what images do, what text does, what captions do, and how sequence creates meaning.

Read:

- `references/visual-hybrid-playbook.md` for type-specific planning
- `references/page-spread-plan.md` for page architecture
- `references/image-caption-system.md` for image research and caption logic

Use `high-stakes-book-review` when visual material involves permissions, real people, brands, children, vulnerable groups, safety, or sensitive places.
