# Evidence Thresholds

| claim type | minimum threshold |
|---|---|
| personal experience | frame as personal, do not generalize |
| practical advice | credible source or tested method plus limits |
| historical fact | primary source or corroborated reputable secondary source |
| medical/health | current official/professional or peer-reviewed source plus expert review when behavior may change |
| legal | jurisdiction, current statute/regulation/case or professional source, expert review for action guidance |
| financial | regulator, filing, audited report, official statistics, or professional source; risk and date stated |
| technical/safety | official docs/specs, versioned tested example, failure-mode warning |
| allegation | primary evidence plus corroboration and legal review |
| real-person private fact | consent/public-interest test plus privacy/legal review |

If threshold is not met, soften, source, defer, or remove.

## Domain Escalation

- Health: diagnosis, treatment, medication, child/pregnancy, mental-health, emergency, diet/exercise advice.
- Law: contracts, lawsuits, rights, immigration, taxes, employment, family, criminal exposure.
- Finance: investing, borrowing, insurance, retirement, trading, tax decisions.
- True crime/biography: named living people, victims, minors, private facts, allegations.
- History/social conflict: contested atrocity, identity, territory, religion, colonial/political claims.
- Technical: security, data loss, physical risk, regulated operation.

## Required Output

For each risky claim, produce:

- claim text
- domain
- current source quality
- missing evidence
- possible harm
- expert reviewer type
- action: keep, soften, source, revise, remove, or legal/expert review
