# Project Diagnosis

Ask only what is missing. Prefer inferring from the user's idea when possible.

## Diagnostic Axes

| Axis | Decide |
|---|---|
| Market classification | Where would readers expect to find this book? |
| Knowledge domain | What fields does it depend on? |
| Manuscript form | Fiction, narrative nonfiction, practical, scholarly/technical, visual/hybrid |
| Reader | General, beginner, practitioner, expert, student, child/YA, niche |
| Evidence burden | experiential, reported, practical, documentary, technical, scholarly |
| Research methods | literature, interview, survey, field, archive, data, case, competitor, fact-check |
| Risk level | low, moderate, high, severe |
| Publishing intent | commercial, independent, platform, education, research, internal |
