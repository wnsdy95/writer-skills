# Plot and Character Arc

## Character Fields

- desire: conscious goal
- need: necessary inner change
- wound/false belief
- secret
- pressure point
- choice pattern
- arc endpoint

## Plot Fields

- disruption
- first commitment
- escalation
- midpoint reversal
- crisis choice
- climax action
- resolution consequence

Rule: each major plot beat should pressure a character flaw, and each major character choice should alter the plot.
