# Scholarly Technical Writing

## Workflow

Make expertise legible. Define the reader's prior knowledge, concepts, terminology, argument, evidence, examples, and validation needs before drafting.

Read:

- `references/scholarly-technical-playbook.md` for structure
- `references/argument-map.md` for claims and evidence
- `references/technical-explanation.md` for examples, diagrams, code, formulas, and teaching order
