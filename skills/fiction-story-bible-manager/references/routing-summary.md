# Routing Summary

Use the smallest skill that can produce the required artifact.

- Book idea diagnosis: `book-project-architect`
- Research plan or source quality: `book-research-designer`
- Concept, TOC, chapter cards: `book-structure-designer`
- Drafting, revision, editing passes: `book-drafting-editor`
- Fiction plot, character, scenes: `fiction-book-writing`
- Fiction canon, timeline, setup/payoff memory: `fiction-story-bible-manager`
- Memoir, biography, history, reportage, true crime: `narrative-nonfiction-writing`
- Self-help, business, education, manuals, how-to: `practical-nonfiction-writing`
- Scholarly, technical, scientific, professional: `scholarly-technical-writing`
- Picture books, comics, photo/cook/travel/art books: `visual-hybrid-book-writing`
- Health, law, finance, crime, real people, privacy, vulnerable groups: `high-stakes-book-review`
