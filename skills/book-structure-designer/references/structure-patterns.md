# Structure Patterns

## Fiction

Use scene, plot, and character arc systems. Route detailed fiction structure to `fiction-book-writing`.

## Narrative Nonfiction

Patterns:
- chronological investigation
- braided personal/history structure
- case-centered chapters
- event countdown

## Practical Nonfiction

Patterns:
- problem to promise to method
- step-by-step transformation
- framework plus cases
- diagnose, act, troubleshoot

## Scholarly/Technical

Patterns:
- definitions to concepts to argument
- foundation to system to applications
- problem, prior work, model, evidence, implications

## Visual/Hybrid

Patterns:
- page/spread beats
- image-led sequence
- recipe/process sequence
- field-guide taxonomy
