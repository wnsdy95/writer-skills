# Chapter Card System

Use one card per chapter.

```markdown
## Chapter Card

- chapter_number:
- working_title:
- chapter_job:
- reader_question:
- opening_move:
- core sections/scenes:
- evidence_or_canon_needed:
- example/case/image:
- ending_turn:
- risk_or_continuity_notes:
```

Reject chapters whose job is only "introduce topic". A chapter should change the reader's understanding, skill, emotional state, or narrative question.
