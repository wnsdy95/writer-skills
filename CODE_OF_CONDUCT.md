# Code of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Discuss ideas and implementation choices directly and professionally.
- Give actionable feedback tied to the repository content.
- Respect privacy, attribution, and consent when discussing examples or source material.
- Be especially careful with material involving real people, vulnerable groups, health, law, finance, crime, politics, or sensitive history.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without consent.
- Submitting material that knowingly plagiarizes or violates copyright.
- Using the project to present unverified high-risk claims as professional advice.

## Enforcement

Maintainers may edit, hide, or remove issues, pull requests, comments, and contributions that violate this code of conduct. Repeated or serious violations may lead to being blocked from participation.
